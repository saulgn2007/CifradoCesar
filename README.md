# CifradoCesar

## 🔍Objetivos🔎 
Crear un programa en Java que permita cifrar y descifrar un mensaje utilizando el algoritmo de cifrado César.
## 🎯Requisitos
- El programa debe solicitar un texto y una clave (un número entero que representa el desplazamiento).
- Cada letra del texto original debe desplazarse X posiciones en el alfabeto.
- Si el desplazamiento sobrepasa la 'Z', debe volver a empezar desde la 'A' (aritmética modular).
- El programa debe mantener los espacios y caracteres especiales sin cifrar.
## 💻Tecnologias
Este proyecto ha sido desarrollado en java con el ID Eclipse.
## 🖥️¿Como ejecutarlo?
Para ejecutar este programa deberemos importar el proyecto en Eclipse y lanzar el main.
